# Make 시나리오 공유 링크

이 문서는 제출용 Make 시나리오 공유 링크를 정리하기 위한 문서입니다.

## 1. 메인 뉴스 요약 시나리오

- 공유 링크: https://eu1.make.com/public/shared-scenario/f7oJNhByEAH/%EB%89%B4%EC%8A%A4-%EC%9A%94%EC%95%BD-%EC%9E%90%EB%8F%99%ED%99%94
- 실행 일정: 매일 09:00 (`Asia/Seoul`)
- 설명: AI타임스 RSS 피드에서 로봇 관련 기사를 수집하고, 중복 확인, 원문 본문 수집, 본문 정제, Gemini 요약, Notion 저장까지 수행하는 메인 자동화 시나리오입니다.

## 2. 오류 모니터링 시나리오

- 공유 링크: https://eu1.make.com/public/shared-scenario/oj9Ao6ONbij/%EC%97%90%EB%9F%AC-%EC%B2%98%EB%A6%AC
- 실행 일정: 매일 09:10 (`Asia/Seoul`)
- 설명: Make의 Incomplete executions를 조회하여 재시도 후에도 실패한 실행을 확인하고, 오류 ID 기준으로 중복 여부를 확인한 뒤 Notion에 오류 로그를 저장하는 보조 시나리오입니다.

## 3. Blueprint 파일 안내

각 시나리오의 Blueprint JSON 파일은 아래 폴더에 포함되어 있습니다.

- `Make_시나리오/메인_뉴스_요약_시나리오/`
- `Make_시나리오/오류_모니터링_시나리오/`

Blueprint 파일을 업로드할 경우, API 키, 인증 토큰, Authorization 헤더, 개인 계정 정보, 비공개 URL 등 민감한 정보가 포함되어 있지 않은지 반드시 확인합니다.

## 4. 보안 주의사항

본 저장소에는 Make, Notion, Gemini API 등의 실제 인증 정보는 포함하지 않습니다.  
공유 링크 또는 Blueprint 파일을 제출할 경우에도 민감한 연결 정보가 노출되지 않도록 확인합니다.
